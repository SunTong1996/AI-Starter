# 计算机基础认知讲座

这个项目旨在帮助大学生更好地认识计算机，了解计算机的基本原理、组成部分和发展历程。

## 项目内容

- 计算机基本概念介绍
- 计算机硬件组成详解
- 操作系统与软件概述
- 计算机网络基础
- 数据存储与处理
- 计算机发展历史与未来趋势

## 使用方法

打开index.html文件即可开始浏览讲座内容。

## 项目结构

```
computer-education/
├── index.html    # 主页面
├── style.css     # 样式文件
├── script.js     # 交互脚本

```
